📝 Proyecto Kanban Personal

Descripción: Este proyecto simula un sistema estilo Trello donde los usuarios pueden crear tableros, listas y tarjetas para organizar sus tareas o proyectos personales. Está orientado a demostrar conocimientos de HTML, CSS y JavaScript en una interfaz moderna e interactiva.

🎯 Objetivo del sistema

Facilitar la organización visual de tareas mediante un tablero Kanban que permite:

  • Crear tableros personalizados
  • Gestionar listas (Por hacer, En progreso, Completado, etc.)
  • Agregar y eliminar tareas dentro de cada lista
  • Agregar o eliminar listas completas

🧪 Instrucciones de uso

  1. Accede a index.html para ver la pantalla de inicio.
  2. Desde allí, regístrate o inicia sesión (simulado).
  3. Una vez dentro, crea nuevos tableros personalizados.
  4. Haz clic en “Abrir” para ingresar al tablero.
  5. Agrega nuevas listas en la parte superior del tablero.
  6. Dentro de cada lista puedes agregar o eliminar tareas.
  7. También puedes eliminar listas completas usando el botón ✖.

⚙️ Tecnologías utilizadas

  • HTML5 – Estructura de las interfaces
  • CSS3 – Estilización responsiva y profesional
  • JavaScript – Lógica para interacción dinámica
